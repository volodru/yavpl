in package

SimpleFilesModel.php : 33

Простое хранилище файлов. Использовать как прототип для наследования.

В основе (и является предком) простой словарь SimpleDictionaryModel с целочисленным первичным ключом.

Замечание по наименованиям: должно быть file_name, file_size и т.п. но не filename!

Структура таблицы передается в конструкторе. Поля file_name, file_ext, file_size обязаны присутствовать в таблице и интерпретируются этим классом как свои.
